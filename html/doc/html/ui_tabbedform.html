Descrição:

TODO: TRANSLATION O tabbedform é extendida da classe form class e difere, principalmente na forma como os dados são apresentados. Com o tabbedform, é possível criar páginas nas quais os campos serão apresentados. Vale ressaltar que, quando uma página não está ativa, os campos do formulátio são do tipo hidden.
