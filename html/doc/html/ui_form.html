Classe Form

Descrição:

The form class is the base class to construct a HTML form. Along with this class, different form object classes are defined, such as input fields (TextField), choices (Selection), check and radio boxes.

GetFormValue

Descrição:

Obtains a submitted form field's value. This is a static function which takes in account that the Form class adds a prefix 'frm_' to all form variables. So when calling this function, the caller has to omit the prefix, using the same name, which was used in the field definition
function GetFormValue($name,$value=false)

Parâmetros:

$name
Descrição do parametro $name
$value Default false
Descrição do parametro $value

Retorna:

(mixed) value of field contained in $_POST
SetFormValue

Descrição:

Sets the content of a form field to the specified value. The function does this by setting both, the field's value member and the global $_REQUEST to remain consistency between the values. It is expected, that the caller passes the same name as of the field definition (e.g. w/o 'frm_' prefix).
function SetFormValue($name,$value)

OnSubmit

Descrição:

Adds JavaScript code which is to be executed, when the form is submitted. when the form is generated, and any JS code has been registered using this function, an OnSubmit handler is dynamically generated where the code is placed.

The generated code looks like the following where stmt stands for the registered statments

Parâmetros:

Descrição do parametro $jscode

Exemplo:

function _onSubmit()
{
  return stmt1 &&
         stmt2 &&
         ...
         stmt#;
}

_RegisterField (somente para uso interno)

Descrição:

This function is used internally of the form framework to prepare the form fields for the usage within the form framework.

The function basically renames all fields to 'frm_' + name and assigns the value from the global array $_REQUEST if the field has no value (null) assigned.
function _RegisterField(&$field)

Parâmetros:

&$field (reference)
to a single field or an array of fields If an array of fields is passed, the function is called recursively for each of the contained fields.

Retorna:

(nothing)

ShowHints

Descrição:

This function can be used to show or hide the form's hint texts. Each form element may have a hint text associated with it. Using this method, one can enable/disable the display of the texts. This may be useful for implementing kind of an beginner/expert mode.

GetFieldList

Descrição:

This is a placeholder function to bu the form's field list. It is excpected, that the form returns a scalar list of all defined fields which carry a form field value. Thus, form elements of decorative purpose only should be omitted.

Derived classes such as TabbedForm override this function to provide the list of fields independently of the form's layout.
function GetFieldList()

_SetData (somente para uso interno)

Descrição:

This method sets the form field values in way, that it iterates thru the list of fields and sets the values of all fields matching the data object attribute names.

In short it does the following
$frm_name = $data->name;
This implies that the form field names must be identical the data member names.
