UseComponent

Descrição:

Used to obtain an instance of a component. The function loads the implementation class of the specified component and returns an instance of the same. The component file must be located into the ./classes/components directory
function & UseComponent($name)

Parâmetros:

$name (string)
Nome da classe o modulo que sera acessado

Retorna:

(object) an instance of ThemeElement

Exemplo:

$ticker = $theme->UseComponent('newsticker');
// $ticker->SetTitle('The Latest News');
$theme->AppendContent($ticker);

Classe ThemeElement

Descrição:

The ThemeElement class is the base class for any element which can be put into a theme. Basically the implementation provides a Generate method, which will generate the appropriate HTML output when called.

Derived classes will likely override the Generate method to customize the output.

Atributos:

var $html;
var $class;

Métodos:

Generate, ThemeElement
