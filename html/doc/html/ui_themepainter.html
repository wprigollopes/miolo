Classe ThemePainter

Descrição:

The ThemePainter class is responsible for generating a HTML page containing all elements of the MIOLO Theme class or only those in which the customized theme is interested in. Each theme should provide a subclass of the ThemePainter named '<name>ThemePainter' where <name> represents the theme's name. Examples: UnivatesThemePainter, SlashThemePainter, PhpNukeThemePainter
